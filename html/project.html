Our Mission

Our goal is to develop highly-practical automated software repair technologies for ensuring safety and reliability of software. To this end, we research on fundamental techniques in the field of programming languages with applications to software engineering and software security.

  • Software Error Detection for precisely localizing errors and vulnerabilities in software.
  • Software Verification for rigorously validating the safety and reliability of repaired software.
  • Software Synthesis for efficiently generating patches that can repair software.

Current Research Projects

Software Error Detection

  • ADAPT: Testing Deep Neural Networks with Adaptive Neuron-Selection Strategy
    • ADAPT is an open source white-box testing framework for deep neural networks. The applications of deep neural networks are rapidly increasing. Therefore, the importance of testing deep neural networks is also arising. We present an effective white-box neural network testing took with the adaptive neuron selection strategy.
    [github] [ISSTA 20 paper]

Software Verification

  • NPEX: Null Pointer Exception X
    • NPEX is an open-source patch recommendation system for Java Null Pointer Exceptions (NPE). NPEX aims to help developers to maintain their projects NPE-free by automatically generating compilable patches for NPE.
    [github]
  • Graphick: Learning Graph-Based Heuristics for Pointer Analysis
    • Graphick is a framework to generate qualified graph-based heuristics for cost-effective pointer analysis. Recently, graph-based heuristics have emerged as a trend to develop cost-effective pointer analysis. Designing graph-based heuristics, however, requires a great deal of effort and expertise. To remove the burden, Graphick aims to produce graph-based heuristics automatically. The evaluation results show Graphick successfully generates cost-effective graph-based heuristics.
    [github] [OOPSLA 20 paper]

Software Synthesis

  • SynthRL: Program Synthesis using Reinforcement Learning
    • Program Synthesis was one of the most major problems that computer scientiests tackled. We anticipate newest artificial intelligence is suitable for program synthesis, which was widely known as a complex task that only human could do. In the project SynthRL, we devise a deep learning model, that will synthesize programs only by leveraging program examples (so-called Programing by Examples) with deep learning.
